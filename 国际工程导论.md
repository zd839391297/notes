# 国际工程导论

## 2019/3/24

### DBB模式

- 世行、亚行均采用此模式

- 设计-招标-建造

- 通用模式

- 招标投标制、建设监理制、合同管理制

#### 设计阶段国内外差异

  - 国内特点：施工图设计完成后才招标
  - 国外特点：初步设计完成后招标，施工图设计由承包商完成
  - 差异：施工图设计是否在设计阶段
  - 优点：施工单位不承担设计责任
  - 缺点：设计与施工脱节，保证工程方案可行


- | 可研 | 初步设计     | 技术设计 | 施工图设计   |
  | ---- | ------------ | -------- | ------------ |
  | 方案 | 国外开始招标 |          | 国内开始招标 |

#### DBB模式的优点：
  - 管理方法成熟，各方对有关程序都很熟悉
  - 业主可以自由选择咨询设计人员，可控制设计要求，施工阶段也比较容易提出设计变更，整个设计由业主控制，//设计人员一直在控制，但是业主不一定对设计要求非常明确，设计人员应当引导业主明确设计要求
  - 可自由选择监理人员监理工程
  - 可采用各方均熟悉的标准合同文本（FIDIC文本），双方均明确自身应承担的风险有利于合同管理和风险管理

#### DBB模式的缺点：
  - 设计招标->招标投标->建造的周期较长，以致推迟投产时间，不利于业主资金周转
  - 建筑师/工程师对项目的施工工期不易控制
  - 管理和协调工作较复杂，业主管理费较高，前期投入较高//平行发包
  - 对工程总造价不易控制，特别在设计过程中对“可施工性”考虑不够时，容易产生变更，从而引起较多的索赔 //设计过程中可施工性很差，设计施工严重脱节，65%损失
  - 出现质量事故时，设计和施工双方容易互相推卸责任

#### 各方的风险分配

##### 业主

 - 资金不到位，中国一般业主选择国营承包商
 - 未做好开工前有关准备工作
 - 招标文件拟定的不好
 - 监理工程师不称职//我国设备水平较高，管理水平较差
 - 由于前面的原因，因而工期、投资和质量达不到要求
 - 设计变更频繁//特别大的问题
 - 承包商水平低，不能保证工期和质量//风险管理：风险寻找、风险评估、风险对策
 - 业主方供应设备和材料的风险
 - 承包商、供货商的索赔
 - 通货膨胀
 - 不可抗力//2017FIDIC改为例外事件
##### 承包商

 - 投标时未进行风险分析
 - 投标时报价过低//很典型的情况，中国：三控五管一协调工期最重要；表面上业主赚钱，谈判中的对等原则，但是实际上承包商干不下去时最终损失的还是业主
 - 内部管理水平低
 - 业主资金未到位，支付能力差
 - 工程师的拖延和刁难
 - 分包商的风险：拖期、质量、索赔
 - 供货商的风险：设备材料供应不及时、不合格、缺配件
 - 通货膨胀
 - 没收保函//此处指履约保函，另有支付保函//履约承诺的形式：保函担保（工程主要形式）、现金担保等
 - 带资承包风险
 - 存在包括技术规范、水文、气象、地质等在内的技术风险

#### 什么样的工程适合DBB模式

  - 业主有充裕的建设周期
  - 业主有资金保障
  - 业主有较强的管理能力，特别是协调能力
  - 设计施工分歧较小
  - 项目范围比较明确，不应出现较大变动

#### 项目发包模式影响因素

  - 项目特点

    - 项目范围、工期、复杂性、和合同价格
  - 业主需要

    - 可建造性、价值工程、合同包装和可行性
  - 业主偏好
    - 责任分配、设计控制和授予合同后业主的参与

#### 优点

 - 单个承包商对整个项目负责，有利于在设计阶段预先施工因素，避免了设计和施工的矛盾，可减少由于设计失误引起的变更以及对设计文件解释引发的争端
 - 选定D/B承包商时，设计方案的优劣作为主要的评价指标，可保证业主得到高质量的工程设计在项目初期选定成员，连续性好，项目责任单一
 - 总价包干（可调价），业主可得到早期的成本保证可对分包采用阶段发包方式，缩短了工期，项目可投产，业主能节约费用，减少利息及价格上涨的影响

### 造价

造价=直接费用+间接费用=工程量（由业主需求决定的）×完成单位工程量所需要的资源量（由施工单位技术水平与管理水平决定）×资源单价（由市场决定）+间接费用+利润（通过竞争降低）+风险金（明确工程量、明确工期）

#### 降低承包商利润的方法

- 承包商间竞争

#### 降低风险金的方法

- 降低工程量
- 明确工程量
- 降低/明确工期（水利工程工期较长）

#### 四制

- 项目法人责任制
- 合同管理制
- 监理制

超高层材料价格占到45%，一般建筑材料价格50-60%，机械25-30%，剩下的是人工

#### 造价管理

- 工程范围明确->范围管理
  - 需求范围->不是任由业主需求发展，需考虑业主经济能力与自身工作能力范围
  - 可交付物范围
    - 强制性标准所要求
  - 工程量范围->合同->工程量清单
    - 取决于所采用的合同形式->支付点

#### 项目管理

- 目标管理
  - 功能性目标-本源目标--合同协议书
  - 约束性目标
    - 多长时间
    - 多少钱
    - 功能明细质量划分
  

#### 委托方满意

实际得到与期望得到的相比较

- 管控分母->委托方期望得到的->期望的管理
  - 期望切合实际
- 管控分子
  - 质量
  - 技术
  - 工期

#### 直接成本

- 工程量

  - 由业主需求决定
  - 主观与客观共同决定
  - 客观起主要（核算）
  - 主观也很重要
- 完成单位工程量所需要的资源量
  - 技术水平
  - 管理水平
  - 承包商客观
- 资源单价
  - 市场价格决定
  - 也可以主观
  - 固定总价合同锁定单价->用主观管理方式化解客观
    - 用多少买多少
    - 一次性买入
    - 用期货方法锁定价格（交货期3个月）->一致性建立在功能性上

### 价格合同模式

#### 总价合同

承包商承担工程量变化的风险，承包商风险较高，工程量变化不大

##### 固定总价合同

- 工程量明确
- 时间较短

##### 可调总价合同

- 工程量明确
- 工期较长->初期考虑的式物价影响，现在延申到工程量的调整

#### 单价合同

承包商不承担工程量变化的风险，承包商风险有所降低

##### 纯单价合同

######  固定单价

- 任何情况下，承包商不承担工程量变化的风险
- 双刃剑						

###### 可调单价

- 对超出价格进行调价
- 物价上涨的调价模式

##### 单价+包干模式

- 实务中见得较多
- 部分包干

##### 成本加酬金合同

- 承包商成本全额由业主支付
- 承包商几乎不承担风险
- 工程量变化非常大
  - 可能为全新技术工程
  - 可能时间太紧，没有时间列->灾后工程、战区工程
  - 承包商有专有技术的工程->只有一家承包商能做
- 完成后业主给承包商酬金
  - 利润
  - 上级公司管理费
  - 没有风险金
- 实务中常用成本加百分比酬金，变相鼓励承包商增加成本
- 后来改为成本加固定酬金，没有鼓励承包商降低成本，同时有缺陷，承包商对工程范围可能会提出异议

### 国际工程体系

#### （政府间）框架项目体系

- 中国模式
- 英美体系（中东）
- 法国体系（非洲国家尤其是西非）

#### 争议主要在设计阶段划分问题:

国际工期有重叠，国内一般没有重叠，设计工期较长

英美法国家工期长于我国的原因：场地动员能力差，劳动效率远低于我国

- 中国体系 

  发改委 设计阶段 三个阶段划分：

  - 可研（方案设计）->估算
  - 初步设计 （可能有技术设计//大项目、化工项目） ->概算：控制工程造价最高的标准
  - 施工图设计->预算->与国际惯例有差别，中国一般由设计院设计，对现场把握不好，65%施工与设计脱节损失（DBB中D截至点）

- 法国体系

  - 简明初步设计->对应可研，着眼方案，为决策服务，为了论证项目可行性，结果为文字性报告（DBB中D可能截至在此）
  - 详细初步设计->比国内初步设计要深，很接近施工图设计，只是不涉及施工方案本身，其他全部设计，(大部分项目DBB中的D已经完成)
  - 施工图设计->由施工单位/承包商设计，到施工组织设计水平，比国内施工图设计要深

- 英美体系

  - 方案设计->要求出图，比可研要深，绝大多数已经到达初步设计，为决策服务，决策基于图纸
  - 扩初设计->国内技术设计的全部内容都有，设备表完成（大部分DBB中D截至点）
  - 施工图设计->施工方法给出，设计公司与施工公司交流的桥梁（小部分项目DBB中D截至点）
  - 深化设计->施工公司设计，部分在国内施工图设计中解决

### DB总承包国内与国外区别

- 国内常用平行发包，国际上惯用总包
- 范围上：国际上一般完全总承包，国内一般是土建总承包，其他项目另找
- 设计深度上：中国设计支持到现场施工，易脱节，国际设计一般不深入到现场分工，现场由承包商
- 项目管理程序 ：国内建设行政主管项目要求，国外每个项目单独设计
- 计划管理：国内，施工进度计划管理，国际：设计开始
- 专业分包：国内，竞争较厉害，国际上有一定的垄断
